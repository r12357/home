複素関数

この記事では、多変数の場合は考えず、一変数複素関数について考える。
さて、具体的に複素関数について述べる前に、複素平面の位相について述べよう。
\(z \in \mathbf{C},\ r \in (0, + \infty)\)に対して、\(z\)を中心とする半径\(r\)の開円板(open disc)\(D_{r}(z)\)および閉円板(closed disc)\(\overline{D}_{r}(z)\)を \begin{align*} &D_{r}(z)=\{w\in \mathbf{C}\ |\ |z - w| < r\}\\ &\overline{D}_{r}(z) = \{w \in \mathbf{C}\ |\ |z - w| \leq r\} \end{align*} また \begin{equation*} C_{r}(z) = \{w \in \mathbf{C}\ | \ |z - w| = r\} \end{equation*} とおく。特に\(D_{1}(0)\)を単位円板(unit disc)と呼び、\(\mathbf{D}\)で表す。
\(\Omega \subset \mathbf{C}\)に対して、\(z \in \Omega\)が\(\Omega\)の内点(interior point)であるとはある\(r > 0\)に対して\(D_{r}(z) \subset \Omega\)となることである。\(\Omega\)の内点全体の集合を\(\Omega\)の内部(interior)といい\(\Omega^{i}\)で表す。\(\Omega = \Omega^{i}\)となるとき\(\Omega\)を開集合(open set)であるという。
\(\Omega \subset \mathbf{C}\)に対して、\(z \in \mathbf{C}\)が集積点(accumulation point)(または極限点(limit point))であるとは、ある\(z_{1},z_{2},\cdots \in \Omega\)に対して\(\lim z_{n} = z\)となることである。\(\Omega\)の集積点の全体の集合を、\(\Omega\)の閉包(closure)という。\(\Omega\)の閉包を\(\overline{\Omega}\)で表す。\(\Omega = \overline{\Omega}\)のとき\(\Omega\)は閉集合(closed set)であるという。\(\overline{\Omega} \setminus \Omega^{i}\)を\(\Omega\)の境界(boundary)といい、\(\partial \Omega\)とかく。 \begin{equation*} \Omega が開集合\Leftrightarrow \mathbf{C}\setminus \Omega が開集合 \end{equation*} \(\Omega \subset \mathbf{C}\)が有界(bounded)であるとは、ある\(r > 0\)に対して\(\Omega \subset D_{r}(0)\)となることである。また、\(\Omega\)が有界かつ閉のとき\(\Omega\)はコンパクト(compact)であるという。
証明はしないが次が成り立つ。
定理2
次の3つは同値である。
(1)\(\Omega\)はコンパクト
(2)\(\Omega\)の任意の開被覆に対して、有限個の部分被覆で\(\Omega\)を覆うことができる。
(3)任意の\(z_{1},z_{2},\cdots \in \Omega\)に対して\(\Omega\)の点に収束するような部分列が存在する。
複素平面上の開集合\(U\)が任意の空でない開集合\(V,W \subset U,\ V \cap W = \varnothing\)に対して\(U = V \cup W\)とできないとき\(U\)は連結(connected)であるという。(直感的には開集合で分離できない。つまり、繋がっている)
開集合\(U\)が連結であることと弧状連結であることは同値である。
ここで、\(U\)が弧状連結とは、任意の\(z,w \in U\)に対して、\(\gamma : [0,1] \to \mathbf{C}\)が連続で\(\gamma(0) = z,\ \gamma(1) = w\)を満たす\(\gamma\)が存在することである。
Remark
この\(\gamma\)の定義域を\([a,b]\subset \mathbf{R}\)としてもよい。実際\(\alpha :[0,1] \to [a,b]\)を\(\alpha(t) = at + b(1 - t)\)とすればこれは全単射である。
連結な開集合を領域(region)といい、領域の閉包を閉領域(closed region)という。
定理3
\(\Omega \subset \mathbf{C}\)を開集合とする。このとき、 \begin{equation*} \Omega が連結 \Leftrightarrow \Omega が弧状連結 \end{equation*}

次にやっと複素関数について述べる。
\(\Omega \subset \mathbf{C}\)を定義域とする、複素数値関数\(f : \Omega \to \mathbf{C}\)のことを複素関数という。
複素関数\(f: \Omega \to \mathbf{C}\)が与えられたとき、 \begin{equation*} u(x,y):= \Re f(x + iy),\qquad v(x,y):= \Im f(x + iy) \end{equation*} で定まる実関数\(u,v\)を考えることがある。ここでは\(u,v\)を\(f\)の実部、虚部と呼ぶことにする。

極限と連続性


\(D \subset \mathbf{C},\ c\in \overline{D},\ \gamma \in \mathbf{C}\)とする。任意の正の実数\(\varepsilon\)に対してある正の実数\(\delta\)があって \begin{equation*} 0 < |z - c| < \delta \Rightarrow |f(z) - \gamma| < \varepsilon \end{equation*} となるようにできるならば、\(z \to c\)のとき\(f(z)\)は\(\gamma\)に収束するという。また\(\gamma\)は\(z \to c\)のときの\(f(z)\)の極限ともいい \begin{equation*} \lim_{z \to c} f(z) = \gamma \end{equation*} あるいは \begin{equation*} z \to c\ のとき\ f(z) \to \gamma \end{equation*} とかく。\(z \to c\)のとき\(f(z) \to \gamma \)となるための必要十分条件は\(z_{n} \to z\)なる任意の\(D\)上の複素数列\(\{z_{n}\},\ z_{n} \neq c\)に対し複素数列\(\{f(z_{n})\}\)が\(\gamma\)に収束することである。証明は実関数の場合と同様。この事実から明らかなように次が成り立つ
定理4(Cauchyの判定法)
\(f: \Omega \subset \mathbf{C}\to \mathbf{C},\ c\in \overline{\Omega}\)とする。\(z \to c\)のとき\(f(z)\)が収束するための必要十分条件は、任意の正の実数\(\varepsilon\)に対して、ある正の実数\(\delta\)が存在して \begin{equation*} 0 < |z - c| < \delta,\ 0 < |w - c| < \delta \Rightarrow | f(z) - f(w)| < \varepsilon \end{equation*} となることである。

\(f:\Omega \to \mathbf{C},\ c \in \Omega\)とする。このとき \begin{equation*} \lim_{z \to c}f(z) = f(\lim_{z \to c} z) = f(c)\ \cdots \ (*) \end{equation*} ならば、\(f\)は\(c\)で連続であるという。極限の定義で言えば、任意の正の実数\(\varepsilon\)に対してある正の実数\(\delta\)が存在して \begin{equation*} 0 < |z - c| < \delta \Rightarrow |f(z) - f(c)| < \varepsilon \end{equation*} となることである。
Remark
\(c \in \Omega\)であるので定義域に属する点に対して連続の概念が定義される。たまに\(f(z) = 1/z\)が\(z = 0\)で不連続である。とか言われるが嘘である。
\(f\)を実部\(u\)と虚部\(v\)に分けて考えると \begin{equation*} |f(z) - f(c)| = \sqrt{|u(x,y) - u(a,b)|^2 + |v(x,y) - v(a,b)|^2} \end{equation*} となるから、\((*)\)は \begin{equation*} \lim_{(x,y) \to (a,b)}u(x,y) = u(a,b),\quad \lim_{(x,y) \to (a,b)} v(x,y) = v(a,b) \end{equation*} と同値である。
複素関数\(f:\Omega \to \mathbf{C}\)がその定義域\(\Omega \subset \mathbf{C}\)の任意の点で連続ならば\(f\)は連続であるという。
関数の極限について、実関数と同様に次の法則が成り立つ。
\(f,g : \Omega \to \mathbf{C}\)で\(z \to c\)のとき\(f(z),g(z)\)がともに収束するなら \begin{align*} &\lim_{z \to c}(af(z) + bg(z)) = a\lim_{z \to c}f(z) + b\lim_{z \to c}g(z)\\ &\lim_{z \to c}f(z)g(z) = \lim_{z \to c}f(z)\cdot \lim_{z \to c}g(z)\\ &\lim_{z \to c}\frac{f(z)}{g(z)} = \frac{\lim_{z \to c}f(z)}{\lim_{z \to c}g(z)}\quad (\lim_{z \to c}g(z) \neq 0) \end{align*} したがって、\(f,g\)が連続関数ならばその一次結合\(af + bg\)およびその積\(fg\)も連続関数である。さらに、\(\Omega\)上でつねに\(g \neq 0\)なら\(f/g\)も連続関数である。また、合成関数も連続となる。
\(f(z)\)を\(D\subset \mathbf{C}\)で定義された複素関数とする。任意の正の実数\(\varepsilon\)に対して、ある正の実数\(\delta\)が存在して \begin{equation*} z,w\in D,\ |z-w| < \delta \Rightarrow |f(z)-f(w)| < \varepsilon \end{equation*} となるとき、\(f\)は\(D\)上で一様連続であるという。
定理5
有界な閉集合\(D\subset \mathbf{C}\)で定義された複素連続関数\(f\)は\(D\)で一様連続である。
\(f\)が\(D\)上で一様連続でなかったとすれば、ある正の実数\(\varepsilon\)に対してどんな正の実数\(\delta\)をとっても \begin{equation*} z,w \in D,\ [\ |z-w| < \delta \ ] \ \land \ [\ |f(z)-f(w)| \geq \varepsilon \ ] \ \cdots \ (*) \end{equation*} となる。ゆえに自然数\(n\)に対して \begin{equation*} z_{n},w_{n} \in D,\ [\ |z_{n} - w_{n}| < \frac{1}{n} \ ]\ \land \ [\ |f(z_{n}) - f(w_{n})| \geq \varepsilon \ ] \end{equation*} となる複素数\(z_{n},w_{n}\)が存在する。そして更にこれらは収束列としてもよい。(収束列としない場合はBolzano-Weierstrassの定理より収束部分列を考えるといい。結局変わらないのだが、、)\(\{z_{n}\}\)の極限を\(\displaystyle c = \lim_{n \to \infty} z_{n}\)とすると \begin{equation*} |z_{n} - w_{n}| < \frac{1}{n} \to 0 \quad (n \to \infty) \end{equation*} より\(\displaystyle \lim_{n \to \infty}z_{n} = \lim_{n \to \infty}w_{n} = c\)である。したがって\(f\)の連続性により \begin{equation*} \lim_{n \to \infty}f(z_{n}) = \lim_{n \to \infty}f(w_{n}) = f(c) \end{equation*} となるがこれは\( (*) \)に矛盾する。ゆえに\(f\)は\(D\)上で一様連続である。
